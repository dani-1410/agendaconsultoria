📋 Controle de Alocação

Escolha onde seus dados serão armazenados.

✅ Seus dados ficam no seu Google Drive, com histórico automático de versões e backup garantido.
1
Crie um projeto no Google Cloud
Acesse console.cloud.google.com → APIs & Services → Library → ative a Google Sheets API.
2
Crie um Client ID OAuth 2.0
Vá em APIs & Services → Credentials → Create Credentials → OAuth client ID. Escolha tipo Web application e adicione a origem null (para arquivos locais) ou o domínio onde o app está hospedado em Authorized JavaScript origins.
3
Cole o Client ID e conecte
O app abrirá uma janela de autorização do Google solicitando acesso ao Google Sheets e ao Google Calendar (somente leitura). Após aprovar, a planilha "Controle de Alocacao" será criada automaticamente no seu Drive e os eventos do Calendar de cada consultor serão exibidos na grade.
4
Planilha existente? (opcional)
Se já tiver uma planilha criada, cole o ID dela abaixo para reutilizar os dados.

1
Crie conta no JSONBin
Acesse jsonbin.io → Sign Up.
2
Gere API Key
API Keys → + Create API Key → copie.
3
Cole e conecte
